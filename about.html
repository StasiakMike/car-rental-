ABOUT PROJECT:

  • This is a projet created in pure PHP with SQL DB. Also some JavaScript was used

  • User can review and choose car avalible. Also cars already hired are visible.

  • Booking form allows book the vehicle for maximum 13 days and 23 hours

  • Booking procedure in integrated with PayU payment system. Card, PayPal, money transfers and others are allowed

  • Admin panel dispaly all bookings with informations about car, person, total price and return date
